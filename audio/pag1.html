HTML Basic

HTML (HyperText Markup Language) é a linguagem padrão para a criação e exibição de páginas da web. Aqui você encontrará um resumo das noções básicas de HTML:

Estrutura Básica: Um documento HTML começa com uma declaração !DOCTYPE html que define o tipo de documento. Em seguida, o conteúdo é envolvido em tags html, que contêm duas seções principais: head e body. Tags: As tags HTML são os blocos de construção fundamentais. Elas são delimitadas por < e >.

Existem tags de abertura tag e tags de fechamento /tag. As tags de abertura indicam o início de um elemento e as de fechamento indicam o final. Alguns elementos podem ser auto-fechados, como img. Lembrando que todas essas tags devem estar entre o sinal de maior e menor: < >

Aqui está um exemplo desses comandos: 

                    <!DOCTYPE html>
                    <html>
                    <head>
                        <title>Exemplo HTML</title>
                    </head>
                    <body>
                        <h1>HTML Básico</h1>
                        <h2>HTML Básico</h2>
                        <h3>HTML Básico</h3>
                        <p>Este é um exemplo simples de HTML básico.</p>
                        <img src="exemplo.jpg" alt="Exemplo de imagem">
                    </body>
                    </html>
                

                    <!DOCTYPE html>: Declaração do tipo de documento HTML.
                    <html>: Tag que envolve todo o documento HTML.
                    <head>: Contém informações do cabeçalho, como título da página.
                    <title>: Define o título da página exibido na barra de título do navegador.
                    <body>: Contém o conteúdo visível da página.
                    <h1>: Define um título de tamanho 1.
                    <h2>: Define um título de tamanho 2.
                    <h3>: Define um título de tamanho 3.
                    <p>: Define um parágrafo.
                    <img>: Insere uma imagem na página, com atributos src para o caminho da imagem

Exemplo para inserir imagem:

                        <img src="imagens/exemplo.jpg" alt="Exemplo de imagem">

Lembrando que para que o comando da imagem funcione é necessário que a imagem esteja na mesma pasta onde estão localizados os outros arquivos do site

Styles

Em HTML, os estilos podem ser aplicados de várias maneiras para controlar a aparência e o layout dos elementos na página da web. Aqui estão as principais maneiras de aplicar estilos em HTML:

                <p style="color: blue; font-size: 18px;">Este é um parágrafo estilizado.</p>

Este é um parágrafo estilizado.

Os estilos internos são definidos no cabeçalho do documento HTML usando a tag style.

                <style>
                    p {
                        color: red;
                        font-size: 16px;
                    }
                </style>

Tag ul

Tag ul: é usada para criar uma lista não ordenada. Em uma lista não ordenada, os itens são apresentados sem uma numeração específica ou ordem sequencial. Cada item é representado por um marcador, que por padrão é um ponto.

Exemplo:

                <ul>
                    <li>Item 1</li>
                    <li>Item 2</li>
                    <li>Item 3</li>
                </ul>

  • Item 1
  • Item 2
  • Item 3

Você pode estilizar a lista não ordenada, alterando o tipo de marcador (por exemplo, usando imagens ou estilização CSS), ajustando o espaçamento entre os itens e personalizando outros aspectos visuais para atender às necessidades do seu design.

Tag ol

Tag ol: é usada para criar uma lista ordenada. Em uma lista ordenada , os itens são apresentados em uma sequência específica e são numerados automaticamente. Cada item é representado por um número ou outro tipo de marcador, dependendo do estilo definido.

Exemplo:

                <ol>
                    <li>Primeiro item</li>
                    <li>Segundo item</li>
                    <li>Terceiro item</li>
                </ol>

  1. Primeiro item
  2. Segundo item
  3. Terceiro item

Neste exemplo, temos uma lista ordenada com três itens. Cada item é definido pela tag li. Quando este código é renderizado no navegador, os itens são exibidos em ordem sequencial, com cada item numerado automaticamente.

Tag Table

Tag table: é usada para criar uma tabela que organiza dados em linhas e colunas. É uma estrutura tabular que pode ser usada para exibir informações de forma organizada e fácil de ler.

Exemplo:

                <table>
                    <tr>
                        <th>Nome</th>
                        <th>Idade</th>
                    </tr>
                    <tr>
                        <td>João</td>
                        <td>30</td>
                    </tr>
                    <tr>
                        <td>Maria</td>
                        <td>25</td>
                    </tr>
                </table>

HTML Attributes

Claro! Os atributos HTML são usados para fornecer informações adicionais sobre elementos HTML e para controlar seu comportamento e aparência. Abaixo está um resumo sobre os atributos HTML mais comuns:

  1. Atributos Globais:

    2. São atributos comuns que podem ser usados em praticamente todos os elementos HTML, como por exemplo id, class, style, title, etc.

  2. Atributo Id:

    3. É usado para definir um identificador exclusivo para um elemento. Cada ID em um documento HTML deve ser único, ou seja, não pode ser compartilhado por mais de um elemento na mesma página.

    Exemplo:

                        <div id="meuId"></div>
  3. Atributo Class:

    4. É usado em elementos HTML para atribuir uma ou mais classes a esses elementos. As classes são usadas principalmente para aplicar estilos CSS. Ela também permite que você associe um ou mais nomes de classes a um elemento HTML.

    Exemplo:

                        <p class="destaque">Este parágrafo está em destaque.</p>

    Este parágrafo está em destaque.

    Neste exemplo, a classe "destaque" foi atribuída ao parágrafo usando o atributo class. Isso permite que você aplique estilos específicos ao parágrafo com essa classe, tanto em um arquivo CSS externo quanto em um estilo inline.

  4. Atributo Style:

    5. É usado para aplicar estilos CSS diretamente a um elemento HTML, sem a necessidade de um arquivo de estilo externo ou uma tag style. Com o atributo style, você pode definir propriedades de estilo específicas para um elemento, como cor, tamanho, fonte, margem, etc.

    Exemplo:

                        <p style="color: blue; font-size: 16px;">Este é um parágrafo estilizado.</p>

    Este é um parágrafo estilizado.

    Neste exemplo, aplicamos o estilo diretamente ao elemento p usando o atributo style. Definimos a cor do texto como azul (color: blue;) e o tamanho da fonte como 16 pixels (font-size: 16px;). Qualquer estilo definido dentro do atributo style será aplicado apenas ao elemento em que está definido.

DIV

Tag Div: não possui nenhum significado semântico específico. Ela é usada principalmente como uma caixa de layout genérica para agrupar e estruturar elementos HTML, facilitando a organização do conteúdo de uma página da web.

Ela é frequentemente combinada com outros elementos HTML e estilos CSS para criar layouts mais complexos e estruturados.

Ela é amplamente usada para criar seções distintas em uma página da web, como cabeçalhos, rodapés, barras laterais, contêineres de conteúdo, etc.

Exemplo:

                <div>
                    <h1>Título da PáginaEste é um parágrafo dentro de uma divisão.</p>
                </div>

Neste exemplo simples, a tag div é usada para criar um contêiner que envolve um título de página, um parágrafo e uma imagem. Isso permite agrupar esses elementos relacionados em uma única seção da página, facilitando a aplicação de estilos ou manipulação com JavaScript, se necessário.

Estilizando a Div no Css:

                div {
                    border: 1px solid #ccc;
                    padding: 20px;
                    margin-bottom: 20px;
                    background-color: red;
                    border: 1px solid #ccc;
                    padding: 20px;
                    margin-bottom: 20px;
                    border-radius: 10px;
                    width: 300px;
                    height: 200px;
                    margin-left: 350px;
                }

Neste exemplo de CSS, estamos aplicando estilos a todas as tags div na página. Definimos uma borda, preenchimento e margem para fornecer uma aparência visual agradável e separar as divisões entre si, além de uma cor de fundo, uma borda, tamanho e etc.
